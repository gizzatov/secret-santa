# Secret Santa Bot

## Running
python manage.py run

## Migrations
`python manage.py migrate` - for upgrade scheme
`python manage.py makemigrations some_text` - for generate new migration

## Extra
`python manage.py shell` - runs django like shell with project env

## Service-Dependicies:
- PostgreSQL

## Environment vars
- DB_USER
- DB_PASSWORD
- DB_NAME
- DB_HOST
- DB_PORT
- API_TOKEN
